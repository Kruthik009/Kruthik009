### 📁 Featured Projects

#### 📚 [RAG PDF Chatbot](https://github.com/Kruthik009/RAG-PDF-CHATBOT)
*An intelligent, high-performance Retrieval-Augmented Generation (RAG) system for chatting with multiple PDF documents using Groq AI and HuggingFace.*
* **LLM Engine:** Powered by **Groq AI (Llama 3.3 70B)** for sub-second, context-aware answers.
* **Vector Indexing:** Document chunking and vector embeddings processed locally using **HuggingFace** (`all-MiniLM-L6-v2`) and **FAISS**.
* **Features:** Upload multiple PDFs, single-process indexing, modern Streamlit chat interface, and one-click CSV conversation export.
* **Live Demo:** [https://kruthik-rag-pdf-chatbot.streamlit.app](https://kruthik-rag-pdf-chatbot.streamlit.app/)

#### 🌍 [WanderLust](https://github.com/Kruthik009/WanderLust) 
*A production-ready travel accommodation booking platform designed to connect wanderers with unique stays.*
* **Architecture:** MVC (Model-View-Controller) structure for high maintainability.
* **Security:** Integrated **Passport.js** for user authentication and resource authorization guards.
* **Media Uploads:** Implemented cloud-hosted storage using **Cloudinary** and Multer.
* **Validation:** Structured server-side schema validations utilizing **Joi**.
* **Live Demo:** [wanderlust-ysw7.onrender.com](https://wanderlust-ysw7.onrender.com)

#### 🍲 [Rescue Meal](https://github.com/Kruthik009/rescue-meal)
*A social impact platform minimizing food waste by matching surplus food donors with volunteers and NGOs.*
* **Concept:** Connects restaurants, hotels, and event managers to shelters and food relief networks.
* **Backend:** Dynamic routing and schema design for tracking donor contributions and scheduling pick-ups.
* **Impact:** Focuses on routing edible surplus food safely to those who need it, reducing organic landfill waste.
* **Live Demo:** [https://rescuemeal-frontend.vercel.app](https://rescuemeal-frontend.vercel.app)
